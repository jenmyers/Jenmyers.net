This iteration of my personal website has existed in various versions since 2012. This version (Baymax) was launched in December of 2020. You can see the code of this and previous versions at GitHub.

Tools

I built this site with the static-site generator Jekyll and I deploy it with GitHub Pages.

I use a modified set of icons originally from Flaticon.

Post images are an assortment of my own images, others in the public domain and some fairly used in the context of critical reviews. The default image found on multiple posts is of Betty Broadbent, a "tattooed lady" from the early part of the twentieth century.

I made everything else with HTML, CSS and large amounts of tea.
